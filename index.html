Linguagens

A linguagem de marcação HTML é uma linguagem semântica. Essa linguagem é utilizada para fazer o conteúdo da página: textos, imagens, títulos, etc.

A linguagem de estilo CSS é especializada na aparência. Ela é utilizada para midar cores e fazer o layout do site, o tornando agradável aos olhos.

A linguagem de programação JavaScript é essencial para fazer interações mais complexas. Ela pode ser usada para fazer algoritmos e captar eventos que possam acontecer no site.
